# React + TypeScript + Tailwind v4.1 + TanStack Router + Query

Minimal template to start a modern SPA with:
- React + TypeScript
- TailwindCSS v4.1
- TanStack Router
- TanStack Query
- Vite

Quickstart (uses pnpm):
```bash
pnpm install
pnpm dev      # start dev server
pnpm build    # production build
pnpm preview  # preview build
```

Key points:
- TypeScript-ready structure
- Tailwind v4.1 base config included
- Providers for Router and Query set up
- Small, opinionated starting layout

Suggested structure:
- src/
  - pages/, components/, hooks/, router.tsx, main.tsx
- public/, index.html, tailwind.config.js, tsconfig.json, package.json
